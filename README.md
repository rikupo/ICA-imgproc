# ICAを用いた移動物体の認識
## ICA 独立成分分析プログラム
独立成分分析ICAを行って2枚の画像間で差分画像を作成する．  
RGB画像をグレイスケールに白色化，標準化した後に自然勾配法を使ってICAを行う．    
差分画像へ閾値処理を施し移動物体の領域を推定する．
独立であるかは信号の大小に関係ないため照度が変化した画像でも差分画像を生成できる（はず）
